## Projects

### NetScan — Network Scanner & Analyzer
> Full-stack network scanning tool. Scans your LAN in real time, identifies devices by OS (via TTL), looks up hardware manufacturers from MAC/OUI prefixes, and checks open ports.

**Stack:** Python · Flask · Next.js · TypeScript · SQLite

- Live concurrent scan of 254 hosts using 50 parallel threads
- OS fingerprinting via ICMP TTL analysis
- MAC address lookup via ARP table + OUI manufacturer database (60+ vendors)
- TCP port scanner for 15 common services (SSH, HTTP, HTTPS, RDP, FTP, etc.)
- Export results to CSV · Copy IP with one click · Full scan history

[![Repo](https://img.shields.io/badge/View%20Repo-NetScan-22d3ee?style=for-the-badge&logo=github&logoColor=white)](https://github.com/SalimMahmoodi015/NetScan)

### TaskFlow — Task & Productivity Tracker
> Full-stack task management app with weekly planning, analytics, and keyboard shortcuts.

**Stack:** Next.js · TypeScript · Tailwind CSS · SQLite · better-sqlite3

- Daily and weekly task views with priority levels and categories
- Analytics dashboard — completion rate, 7-day bar chart, category/priority breakdown
- Quick date buttons, keyboard shortcuts (N = new task, 1/2/3 = tabs, Esc = close)
- Full REST API with SQLite persistence
