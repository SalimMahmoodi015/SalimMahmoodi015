## 📦 Projects

### 🌐 NetScan — Network Scanner & Analyzer

> Built because "what's on my network?" shouldn't require a networking degree.

Scans your entire LAN in seconds and identifies every device by OS, hardware vendor, and open ports — all through a clean web UI instead of a terminal dump.

**Stack:** `Python` `Flask` `Next.js` `TypeScript` `SQLite`

- 🔍 Scans 254 hosts concurrently with 50 parallel threads
- 🖥 OS fingerprinting via ICMP TTL analysis (Windows / Linux / macOS)
- 🏭 MAC → manufacturer lookup across 60+ vendors via OUI database  
- 🔓 Port scanner for 15 services — SSH, HTTP, HTTPS, RDP, FTP, and more
- 📤 CSV export · one-click IP copy · full scan history

[![Repo](https://img.shields.io/badge/GitHub-NetScan-22d3ee?style=for-the-badge&logo=github&logoColor=white)](https://github.com/SalimMahmoodi015/NetScan)

### ✅ TaskFlow — Task & Productivity Tracker

> A task manager that shows you how productive you actually are.

Full-stack task tracker with a built-in analytics dashboard — not just a to-do list, but a system for understanding your own work patterns over time.

**Stack:** `Next.js` `TypeScript` `Tailwind CSS` `SQLite` `better-sqlite3`

- 📅 Daily and weekly task views with priority levels and categories
- 📊 Analytics dashboard — completion rate, 7-day bar chart, category breakdown
- ⌨️ Keyboard-first — `N` new task · `1/2/3` switch tabs · `Esc` close
- 🔌 Full REST API with persistent SQLite storage

[![Repo](https://img.shields.io/badge/GitHub-TaskFlow-6366f1?style=for-the-badge&logo=github&logoColor=white)](https://github.com/SalimMahmoodi015/TaskFlow)
